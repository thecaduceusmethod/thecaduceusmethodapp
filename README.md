# The Caduceus Method App

Breathing exercises for vagus nerve stimulation, heart-brain coherence, and frequency-based practice.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev -- --host
   ```
3. Open the URL shown by Vite in your browser.

## Project structure

- `index.html` — application entry point
- `src/main.jsx` — React application bootstrap
- `src/App.jsx` — main app component
- `src/assets` — placeholder image assets for local imports
- `vite.config.js` — Vite configuration

## Available scripts

- `npm run dev` — start local dev server
- `npm run build` — produce production build
- `npm run preview` — preview production build locally
